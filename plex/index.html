Plex Plugin

Plex Plugin is a plugin designed for use with Plex.

With Plex Plugin, you can stream your anime collection to any device that supports Plex allowing you to always have access to your collection no matter where you go. Like the My Anime 3 plugin for MediaPortal, Plex Plugin is capable of displaying your collection data, including images in a detailed and easy to navigate layout providing you the best experience possible.
