Important Information

  • .NET Framework 4.5 is required, first time users and those upgrading from an older version will need to first download and upgrade .NET Framework. You can Download .NET Framework 4.5 from Microsoft.
  • If you are upgrading from version 3.3 or lower, you'll be unable to run JMM Server after upgrading due to changes in the config file. To fix this download Config Fixer and run it.
