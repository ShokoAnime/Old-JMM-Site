Latest Downloads

Here you will find the latest stable versions of JMM as well as beta versions to test out new features.

Important Information

  • .NET Framework 4.5 is required, first time users and those upgrading from an older version will need to first download and upgrade .NET Framework before installing the latest version of JMM. You can Download .NET Framework 4.5 from Microsoft.
  • If you are upgrading from version 3.3 or lower, you'll be unable to run JMM Server after upgrading due to changes in the config file. To fix this download Config Fixer and run it.

Manual Update

If for some reason the automatic update is not working correctly for you or you simply want to upgrade JMM to a specific version, you can do that by using the links below. There's two things you need to pay close attention to when manually updating, the current version you have (also known as the base version) and the version you want to upgrade to. Additionally at the bottom of the list you'll see generic update all files that can be used to update your version of JMM to a specif version, in this instance the base version does not matter.

  • Manually Update JMM Server
  • Manually Update JMM Desktop

Beta Release

The available beta is updated semi-regularly to include all the latest changes from the GitHub source. This means there will be times, especially when a new version is released that the current available beta is either an older build of the current stable release or even an outdated build.

No Beta Release is available at this time.
