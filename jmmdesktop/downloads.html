Downloads Tab

The downloads tab allows you to expand your anime collection by downloading new series and missing episodes using the built-in search and browse feature. In addition, JMM Desktop supports uTorrent integration allowing you to add new torrents and manage existing torrent even if they are a different computer.

At this time, uTorrent is the only supported client.

uTorrent

For uTorrent integration to work you must first enable the Web UI in uTorrent, it should look something like the following.

Once down, click the Settings section and input the Username, Password and Port you inputted in uTorrent. For the Server, input your Computer's Name or the IP Address.
