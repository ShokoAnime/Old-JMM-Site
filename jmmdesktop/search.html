Search Tab

The Search tab is an expanded version of the search function that's available for use in the Collection tab. However like the search function on the Collection tab, it's limited to single word searching.

You have two different search filters you can use, Everything and Title Only.

Option Description
Everything Using this filter will return results with your search term from group name, series name, tags and custom tags
Title Only Using this filter will return results that have your search term only in the series name.
