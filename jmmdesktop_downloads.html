JMM Desktop - Downloads Tab

The downloads tab allows you to expand your anise collection by downloading new series and missing episodes using the built-in search and browse feature. In addition, JMM Desktop supports uTorrent integration allowing you to add new torrents and manage existing torrent even if they are a different computer.

Currently at this time JMM Desktop only supports uTorrent. Additional clients have been brought up and will hopefully be implemented at a later time.

For uTorrent integration to work you must enable the Web UI in your client. It should look similar to the following.

Use the same details you entered in uTorrent for the settings in JMM Desktop.

Once done click Test And Save. If you encounter any problems go check to make sure your settings are correct and you can access the WebUI outside of JMM Desktop.
