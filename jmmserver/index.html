JMM Server

JMM Server is the heart of JMM and is a Mandatory Install. It's responsible for all the behind the scenes work you don't have to do. JMM Server performs all the file hashing, file comparing, information compiling and image downloading and stores it all in it's shared database. With JMM Server, you can run multiple clients such as JMM Desktop or My Anime 3 across multiple computers and always keep your information centralized.
