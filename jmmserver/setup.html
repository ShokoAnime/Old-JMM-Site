Setup Tab

The setup tab allows you to manage and if necessary make changes to your database settings. Please note that changing any information will reset your collection forcing JMM Server to re-hash and download information for all your files. The only thing that will not have to be re-downloaded are your images, unless you manually deleted them or changed the location of the image directory.

Database Type: The type of database JMM Server will use, thee are three options. SQL Server, MySQL and SQLite.
Server Address: The address of your database. By default it's localhost.
Database Name: The name of your database. By default it's JMMserver.
Username: The name of the user connected to the database. By default it's root.
Password: The password you setup for the user to connect to the database.
