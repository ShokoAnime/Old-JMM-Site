Settings Tab

The Settings tab allows you to configure certain settings for JMM Server.

Server Port - This is the port JMM Server will use, by default it's set to 8111.
Images Path - The path for any images download by JMM Server.
Minimize On Startup - If selected, will start JMM Server minimized.
Allow Multiple Instances - If selected, will allow you to run multiple instances of JMM Server.

To save space, make the Images Path of JMM Server and JMM Desktop the same.

Make JMM Start With Windows

While there's no built-in option for this, with a few edits to your registry you can make JMM Server start with windows.

Windows XP, Vista, 7, 8 and 8,1

Press the Windows Key + R to bring up the run command and type in %appdata% and press enter. Now go to \Microsoft\Windows\Start Menu\Programs\Startup, once there open a new window and go to the location where you've installed JMM Server. Right-click on it and create a shortcut and drag that shortcut in the Startup folder we accessed moments ago. Now restart your computer to make sure it worked.

Windows 10

Below you'll find a regkey you can add to make JMM Server start with Windows. First copy the example below into a next text document, then change the path to the location of JMM Server on your computer. The easiest way to do this is by navigating to where you've installed JMM Server and copying the location from the location bar. Replace JMM SERVER LOCATION GOES HERE with the actual location of JMM Server. Before you can move on you need to make a couple changes, below is an example of the correct setup for the location.

"JMMServer"="\"E:\\Program Files (x86)\\JMM\\JMM Server\\JMMServer.exe\""

Make sure you inputed location looks like the one above, double-backslashes and all. Once you're done save the file but instead of saving it as a .txt file, save it as a .reg file instead. Now double-click the file to run it, if any prompts come up click OK to continue, once down you'll be informed that regkey was added successfully. Restart your computer to test and see if it worked, if not then double-check your reg file for any mistakes.

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Run]
"JMMServer"="\"JMM SERVER LOCATION GOES HERE""
