Contributing To JMM

The reason JMM was moved to GitHub was to allow anyone the opportunity to help contribute to the project. We actively encourage anyone to help and to submit any changes they think would benefit JMM. Currently there are six projects being hosted on GitHub under the JMM name that you can help contribute too.

Even simply submitting bugs or requesting new features is contributing to the project. If you believe you've found a problem or have an idea on how to improve JMM then let us know!

Contributing To JMM Suite

It's extremely easy to start contributing to the JMM suite. Simply register for a GitHub Account. For those who already have a GitHub account either fork the project to start working on it or submit an issue if you've found a bug or would like to request a feature. We recommend Source Tree for those looking for a desktop client to manage their GitHub projects. It allows you to upload your commits allowing you to submit a pull request to have your changes merged into the main JMM projects.

Contributing To JMM Website

As this is a volunteer project, meaning everyone involved works on JMM during their free time, the JMM Staff prioritize working on the JMM suite. This means that the website can go unchanged for periods of time with the exception of the downloads and changelog pages. In order to help facilitate development on the website we've created a to-do list of missing pages and pages that need updating. In addition we've written a guide to help make it as easy as possible to contribute to the website. You can find both links below.
