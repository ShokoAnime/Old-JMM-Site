Frequently Asked Questions

Can Anime Buddy connect to JMM Server over the internet?

You bet it can!

However there are a few things you'll need to keep in mind when remotely connecting to JMM Server. First, all metadata and images will temporarily download for any series being displayed in Anime Buddy. Download speeds will depend entirely on your connections download speed and the upload speed of the connection JMM Server is using. You can still view your entire collection even if it's associated metadata and images have not been downloaded yet, you'll just see a placeholder for it until it's been downloaded.

Setting Anime Buddy up to access JMM Server over the interent is easy. Anime Buddy can also remotely playback files in your collection!

JMM Server Not Responding Errors

When attempting to connect to JMM Server, you may receive an error message that's similar to the following:

JMM Server is not responding..

This is a generic error message that is called when Anime Buddy times out when attempting to connect to JMM Server.

Here's are the most likely reasons:

  • Check to make sure JMM Server is running.
  • Check to make sure JMM Server has been configured correctly and the settings in Anime Buddy match.
  • If every thing's configured correctly and you're trying to access JMM Server on the same machine it's most likely a loopback issue.

In regards to #3, this is a common issue where Windows Store Apps are forbidden from sending network traffic to the local machine. You can over-ride this (and get a full explanation) from the following page:

Windows 8 - Apps enable loopback network isolation

Simply install the loopback utility and add Anime Buddy as an exemption and retry connecting to JMM Server.

Playing An Episode/Video

Anime Buddy has the ability to play local files as well as remotely download files from your collection if you've connected Anime Buddy to JMM Server. Setting Anime Buddy up for remote playback is easy and will work on any PC running Windows 8 or newer.

Why Won't The Episode Play?

If you are running Windows RT, you can only play videos which are supported by Windows Store Apps. At the time of writing the Video App only supports playback of AVI and XVID files.

For an alternative player we recommend VLC. Users On Windows 8/8.1/10 can download alternative media players like Media Player Classic, VLC or PotPlayer, all which are supported by JMM.
