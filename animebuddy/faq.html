Frequently Asked Questions

Can Anime Buddy connect to JMM Server over the internet?

Yes it can!

However there are a few things to keep in mind when connecting remotely to JMM Server. All data (images and information) will temporarily download for the series being displayed. Depending on your internet connection, this can take a while causing an image outline to appear in the places of images and blank information boxes. JMM Server also handles the downloading of your media which again depending on your internet connection can take awhile to load.

Information for how to do this can be found in the Anime Buddy setup section.

JMM Server Not Responding Errors

When attempting to connect to JMM Server, you may receive an error message that's similar to the following:

JMM Server is not responding..

This is a generic error message that is called when Anime Buddy times out when connecting to JMM Server.

Here's are the possible reasons:

  • 1. JMM Server is not running.
  • 2. You have not entered the JMM Server details correctly.
  • 3. Loopback issues when trying to connect Anime Buddy to JMM server on the same machine.

In regards to #3, this is a common issue where Windows Store Apps are forbidden from sending network traffic to the local machine. You can over-ride this (and get a full explanation) from the following page:

Windows 8 - Apps enable loopback network isolation

Simply install the loopback utility and add Anime Buddy as an exemption and retry to connect to JMM Server.

Playing An Episode/Video

To play an episode from your collection in Anime Buddy, the file must first be downloaded. The reason for this is that Windows Store Apps are not allowed to access the network, or local machine folders without manually using the File Picker. When the file is done downloading it will immediately begin playing, additionally downloaded files are not deleted automatically allowing you to re-watch them. However if you'd like to delete your temporary downloads click Clear Temp Downloads on the episode summary window or Delete Cached Video Files in the options menu.

Why Won't The Episode Play?

If you are running Windows RT, you can only play videos which are supported by Windows Store Apps. At the time of writing the Video App only supports playback of AVI and XVID files.

For an alternative player we recommend VLC. Users On Windows 8/8.1/10 can download alternative media players like Media Player Classic, VLC or PotPlayer, all which are supported by JMM.
