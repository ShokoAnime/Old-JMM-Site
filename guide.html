Website Guide

Thank you for volunteering to help out with the website, we truly appreciate it!

This guide was written to make it easier for anyone to contribute to the website regardless of skill level. Below you'll find code examples and instructions on how to add different functionality present in this website to the page you are working on.

Couple notes before you continue.

  • All available functionality is listed, if you'd like additional functionality added create a issue on the GitHub page.
  • Only resize an image if needed, also you do not need to create a thumbnail.
  • This guide does not cover the basics of HTML, head over to W3Schools if you need help with that.

Displaying Code

This layout uses Google's Prettifier that allows syntax highlighting in HTML which is useful for showing bits of code. Since it's used for showing all the examples on this page, we'll cover it first.

Code 

<pre class="prettyprint">Your code bit goes here.</pre>

Creating A Link To The Menu

If you want to have a link on the menu you need to assign the id tag to the links location. The id tag is usually assigned to a span class tag above the link title, for example to an h3 as the header tags are used to break up the content on your page. When creating the id for the link use a hyphen "-" between words, also do not create long link names. Move down this page a bit and click Creating A Link To The Menu from the menu to be taken back here.

Code

<span class="anchor" id="link-to-menu"></span>
<h3>Creating A Link To The Menu</h3>

Creating A Drop Down Menu

If you're creating a page that has more than 15 links, you'll find that the scroll menu won't display all the links. You can get around this by creating a drop down menu.

The drop down menu and its contents are contained within the <div class="click-nav"> class so just copy the code starting there and ending at </div>. Just place that code where you want your drop down menu and you'll all set. For a live example check the JMM Desktop Utilities Tab page.

      <div class="click-nav">
        <ul class="no-js">
          <li> <a href="#" class="clicker">Unrecognized</a>
            <ul>
              <li><a href="#unrecognized-overview">Unrecognized Overview</a></li>
              <li><a href="#managing-unrecognized-files">Manging Your Unrecognized Files</a></li>
              <li><a href="#linking-unrecognized-files">Linking Your Unrecognized Files </a></li>
            </ul>
          </li>
        </ul>
      </div>

Adding Lightbox To Images

Lightbox is required for all images you display on your page. Add data-lightbox="data-name" and data-title="Data Name" to the image's link. Best practice is to use the image name as the data-name and link alt as the data name. The code below is for the JMM Desktop image on the index page.

Code

<a href="images/jmmdesktop/metro/metro-dashboard-01.jpg" data-lightbox="metro-dashboard-01" data-title="JMM Desktop - Metro Dashboard"><img src="images/jmmdesktop/metro/metro-dashboard-01.jpg" alt="JMM Desktop - Metro Dashboard"></a>

Creating Alerts

There are four different types of alerts that can be displayed depending on the need, alerts should not be overused. An example can be see on the Installing and Setting Up MySQL page.

error:

success:
Write your success message here.
warning:
Write your warning message here.
notice:
Write your notice message here.

Adding Tabs

Tabs are beneficial when you have a lot of content to display for a particular section. You can have as many tabs as you want though you should break up the section if you have more than 5 tabs. Linking to another tab within a tab is supported, however linking to a tab from either another page or from another part of the site is not currently supported.
