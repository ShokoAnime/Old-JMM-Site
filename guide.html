Website Guide

Thank you for volunteering to help out with the website, we truly appreciate it!

The purpose of this guide is to make it easier for anyone that's wants to contribute to the website and provide the code for specific functions that are used on the website. With each code example you'll find detailed instructions on implamenting the code and an example to see how it looks.

Some things to keep in mind before continuing on.

  • All available functionality is listed, additional functionality or an alternative to a current function can be requested by creating an issue on GitHub.
  • Images are automatically scaled, no need for resizing or creating thumbnails.
  • This guide is only meant to explain the functionality used on this site, if you're looking for information on coding, check out W3Schools.

Displaying Raw Code

This layout uses Google's Prettifier that allows syntax highlighting in HTML which is useful for showing raw code. Since it's used for showing all the examples on this page, we'll cover it first.

Code 

<pre class="prettyprint">Your code bit goes here</pre>
Simply add the above to the page and input the code you want to show by replacing the Your code bit goes here part. If your editor provides a code and design view of the page, insert the code in code view and then input the code you want to show in the design view.

Creating An Internal/External Link Using ID Tags

Notice how when you scroll down the page, the location on the menu changes to match your current location or if you click a link to another page it takes you to an exact spot? This is done by creating a link to a specific ID tag which you can use in multiple ways.

If you're creating a link on the same page, you can do it by simply linking to the ID using the following code.

Code

<span class="anchor" id="internal-external-link"></span>

Put this code above your link's location. Avoid creating long link names and use hypens "-" to sperate words, letters or numbers, using anything else causes problems,. Here's what the code looks like for the link to this section.

Code

<span class="anchor" id="internal-external-link"></span>
<h3>Creating An Internal/External Link Using ID Tags</h3>

Scroll down a bit and then click on Creating An Internal/External Link Using ID Tags from the menu on the right-side to be taken back to this section.

If you're linking to another page its a very similar setup

Code

<a href="index.html#jmm-desktop">JMM Desktop</a>

In this example, the link will take the user directly to the JMM Desktop part on the index page. The setup is your normal link setup with the ID after the .html part which in this case is #jmm-desktop.

Creating A Drop Down Menu

If you're creating a page that has 10 or more links, the menu will not display correctly and some of your links will not be accesible. This is because the menu scrolls with the user and a long menu will not scroll. To get around this you can incorporate a drop down menu into your menu structure using the following code. 

      <div class="click-nav">
        <ul class="no-js">
          <li> <a href="#" class="clicker">drop down menu activator</a>
            <ul>
              <li><a href="#link1">Link 1</a></li>
              <li><a href="#link2">Link 2</a></li>
              <li><a href="#link3">Link 3/a></li>
            </ul>
          </li>
        </ul>
      </div>

The drop down menu and its contents are contained within the<div class="click-nav">as shown above. The link with the clicker class is the name of drop down menu with the links in the unordered lists below it. Just place that code where you want your drop down menu and you'll all set.

For a live example check the JMM Desktop Utilities Tab page.

Adding Lightbox To Images

Lightbox is required for all images you display on your page, don't worry though as it's extremely easy to implament. Simpy adddata-lightbox="data-name" and data-title="Data Name" to the image's link. Best practice is to use the image name as the data-name and link alt as the data name.

The code below is for the JMM Desktop image on the index page.

Code

<a href="images/jmmdesktop/metro/metro-dashboard-01.jpg" data-lightbox="metro-dashboard-01" data-title="JMM Desktop - Metro Dashboard"><img src="images/jmmdesktop/metro/metro-dashboard-01.jpg" alt="JMM Desktop - Metro Dashboard"></a>

You can also link images together by making data-lightbox="data-name" the same for each image.

Creating Alerts

There are four different types of alerts supported that can be displayed depending on the need. Alerts should be used sparingly as they are not meant to be used as indicators or seperators.

An example can be see on the Installing and Setting Up MySQL page.

Adding Tabs

Tabs are beneficial for keeping your page length from becoming too long. The max amount of tabs you can have per section is five tabs. Linking to another tab within a tab is supported, as well as directly linking to a tab on another page.

You can have up to four tabs per page. Simply change the div id from tab-01 to tab-02,tab-03 or tab-04.

Example

  • Tab 1
  • Tab 2
  • Tab 3

Tab 1

Tab 2

Tab 3

Code

      <div id="tab-01">
        <ul>
          <li><a href="#tab-1">Tab 1</a></li>
          <li><a href="#tab-2">Tab 2</a></li>
          <li><a href="#tab-3">Tab 3</a></li>
        </ul>
        <div id="tab-1">
          <p>Tab 1</p>
        </div>
        <div id="tab-2">
          <p>Tab 2</p>
        </div>
        <div id="tab-3">
          <p>Tab 3</p>
        </div>
      </div>
