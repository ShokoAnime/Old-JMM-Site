JMM Installation and Setup

Installation and Setup

JMM Server and JMM Desktop are mandatory installs, everything else is optional

  • Step 1 - Install JMM Server
  • Step 2 - Install JMM Desktop
  • Step 3 - Install My Anime 3 / Plex Agent / Anime Buddy

NOTE – JMM Server and Desktop include automatic update functionality, and will inform you if an update is available.

Running JMM Server the First Time

JMM supports Microsoft SQL Server 2008 (and higher), MySQL and sqlite as the database back-ends. The first time you start JMM Server you will be prompted to select a database

We recommend that you use SQL server or MySQL instead (especially if you have over 200 series in your collection)

Selecting a Database

The default is sqlite, which is the easiest to use and doesn’t require any setup from the user. For those who want to better performance you can read more info about SQL Server and MySQL.

The links will also give you information on how to install and configure those database services

Configuring JMM Server

After selecting and configuring your database, you should now see extra tabs in JMM Server

This means you are ready to enter your AniDB and other essential settings.

The best way is to run ‘JMM Desktop’ and edit your settings there.

Running JMM Desktop the First Time

If you are running JMM Desktop on a different machine to JMM Server than you will probably see the following screens.

Enter the address or name of the machine where JMM Server is running and press the ‘Test and Save’ button.

If it connects successfully (or you are running Desktop and Server on the same machine) you will see the login prompt.

When your JMM server initially setup your database it created two users. ‘Default’ and ‘Family Friendly’ (both with blank passwords)

  • Default - Has all your content
  • Family Friendly - Has hidden content based on the categories/genres that are not family friendly

You can edit this on the ‘Users’ section of the ‘Settings’ tab. Log in and go to the ‘Settings’ tab. The ‘Essential’ section is what we are interested at this time.

Enter your AniDB username and password. If you don’t have an AniDB account you will need to signup.

Then start adding your Import Folders.
Basically these are the locations of the videos you want to import (sub folders will be included automatically)
If JMM Desktop is on a different folder to JMM Server you will need to enter the path as it is on the server, not on the client.
If you are in this situation the section below on mapping import folders on remote clients.

At this stage you can either have a look at the other less important settings or go directly to the ‘Server’ tab

Press the ‘Run Import’ button to start processing your files.

This can take a long time depending on how many files you have. However you can go to the ‘Collection’ tab to see the series/episodes/files as they are imported.

Post Import Completion

Once you have finished importing your collection, you can optionally choose to download your votes from AniDB

This is available from the Actions tab on the JMM Server, or the Server tab on JMM Desktop
