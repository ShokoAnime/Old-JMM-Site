Japanese Media Manager

Installation and Setup

JMM Server and JMM Desktop are mandatory installs, everything else is optional. Both JMM Server and JMM Desktop support automatic updating, when an update is available the program will let you know.

  • Installing And Setting Up JMM Server
  • Installing And Setting Up JMM Desktop

Installing And Setting Up JMM Server

Run the downloaded .exe file and follow the steps to install JMM server. Once you're done start JMM Server, you'll be prompted to select a database type.

JMM supports Microsoft SQL Server 2008 (and higher), MySQL and SQLite as the database back-ends. The first time you start JMM Server you will be prompted to select a database. For large collections, it's recommend that you use either SQL Server (2008 and above) or MySQL instead of SQLite.

Selecting a Database

The default database type is SQLite, which is the easiest to use and doesn’t require any setup from the user. For those who want a better and faster performance you can read more info about installing and using SQL Server and MySQL.

Configuring JMM Server

After selecting and configuring your database, you should now see five new tabs in JMM Server.

Installing And Setting Up JMM Desktop

If you are running JMM Desktop on a different machine to JMM Server than you will probably see the following screens.

Enter the address or name of the machine where JMM Server is running and press the ‘Test and Save’ button.

If it connects successfully (or you are running Desktop and Server on the same machine) you will see the login prompt.

During JMM Server database setup, your database created two users. Default and Family Friendly, by default each user's password is left blank.

  • Default - Will display all your content.
  • Family Friendly - Has hidden content based on the categories/genres that are not family friendly

You can edit the default users by clicking the Settings tab and then the Users tab.

Click the Settings tab then the Essential tab. The Essential tab is where all information required for JMM Server and JMM Desktop to work is stored.

First, enter your AniDB username and password. An AniDB account is required for JMM Server and JMM Desktop to work, Click Here to sign up. Make sure you click Test and Save to make sure your inputted AniDB information is valid.

Now setup your Import Folders, you just need to add the root folder of the location where you keep your anime. All sub folders are automatically added. If JMM Desktop is on a different computer than JMM Server you will need to enter the path as it is on the server, not on the client. The screenshot below has an example of a mapped server path.

You can ignore the other settings for now, for the last part click the Server tab.

The Server tab shares a lot of options with JMM Server, so if you need to make changes you can do it via JMM Server as well.

Click the Run Import button, this will start scanning and adding your anime into your collection. Depending on the amount of files that need to be scanned this process could take a very long time. JMM Desktop does not need to be running as the importing process is handled by JMM Server. However if you'd like to view your collection while it's importing click on the Collections tab to see what's been imported so far.

Post Import Completion

Once you have finished importing your collection, you can go back to the Server tab in JMM Desktop or open JMM Server and run some additional tasks such as syncing votes, collection status and updating media info.
