What Is JMM

JMM (Japanese Media Manager) is a series of programs and plugins that are ideal for any anime fan looking for a way to catalog their anime collection regardless of size without the hassle of manually inputting each and every file. In addition, the JMM suite provides the user with the most flexibility in managing their collection requiring minimal input from the user allowing the user to enjoy their collection.

JMM (Japanese Media Manager) is an anime catalog program that automatically catalogs your anime collection regardless of size and without the hassle of having to manually input each and every file in your collection. In addition, JMM provides you with the most flexibility in managing their collection requiring minimal input from the user allowing you to enjoy their collection.

JMM is currently composed of the following programs and plugins.

  • JMM Server - Shared server component
  • JMM Desktop - Windows application
  • My Anime 3 - A plugin for MediaPortal (Home Theatre PC's)
  • Plex Plugin - A plugin for Plex (Home Theatre PC's)
  • AnimeBuddy - A Windows Store app for Windows 8/8.1/10

How does JMM work?

Information from various sites such as AniDB, TheTvDB and TMDb is gathered and complied to deliver the best anime experience possible. The goal of JMM is to reduce the amount of work you have to do by making everything as automated as possible. This is possible thanks to AniDB and it's vast anime database as AniDB is used by most subbing groups as an information repository, which then allows JMM to use this information for your collection. This allows JMM to not have to rely on your files being named in a specific format which results in your files being recognized, cataloged and ready for playback with no additional work on your part. JMM also provides tools to identify and catalog your files in those rare instances where it could not be done automatically.

While there are other programs out there with similar functionality, JMM differs from them as it uses a client-server based architecture called JMM Server. This means you have one central server running which keeps track of your collection, while clients like JMMDesktop, MyAnime3 and Plex Plugin connect to the server to display your collection.

Metadata from various data gathering sites such as AniDB, The TV Database and The Movie Database is gathered and complied for use in JMM to add detailed information about the files and series in your collection which in turn allows JMM to deliver you the best anime experience possible. JMM is able to compare your files without having to rely on the filename by comparing it against AniDB's database which results in your files being recognized, cataloged and immeditly ready for playback with absolutely no additional work on your part. While there are other anime catalog programs out there that provide similar functionality, there's nothing else like JMM available for use as JMM uses a client-server based architecture as it's base while other porgrams just use a client based architecture. What this means is that JMM is more than just a single program or plugin. Using JMM Server, the heart of JMM provides access to your entire collection across different clients such as JMM Desktop and Anime Buddy and media center programs like Media Portal via My Anime 3 and Plex via Plex Plugin. It doesn't matter if you're at home or away, as long as you have an internet connection, your collection is never more than a few clicks away!

Key Features

Here are some of the key features JMM provides.

  • One server allowing for multiple clients
  • Store your anime on internal and external media, JMM complies the information into it's database
  • Create multiple profiles each with their own collection and stats
  • Advanced filtering and sorting options for displaying and updating your collection
  • Supports popular sites AniDB, The TvDB, MovieDB, MyAnimeList and Trakt.tv
  • Supports SQLite, MySQL or Microsoft SQL Server for your database
  • Direct integration with uTorrent and multiple torrent trackers (Anime Byt.es, BakaBT, Nyaa and Tokyo Toshoka)
  • Support for advanced file management such as file renaming, missing episodes check, duplicate files check, manual linking, AVDump and more
  • Ability to generate recommendations based off your ratings
  • For Trakt.tv users, see what your friends are watching

JMM Server

JMM Server is the heart of JMM.

It's responsible for all the behind the scenes work you don't have to do. JMM Server performs all the file hashing, file comparing, information compiling and image downloading and stores it all in it's shared database. With JMM Server, you can run multiple clients such as JMM Desktop or My Anime 3 across multiple computers and always keep your information centralized.

Learn more about JMM Server

JMM Server is the heart of JMM and it's client-server architecture design.

JMM Server handles all the behind the scenes work automating it all so you don't have to manually process your collection. JMM Server performs file hashing, file comparing, information compiling and image downloading for each file you add and it's corresponding series. JMM Server is what allows you to have multiple clients such as JMM Desktop and Anime Buddy and media center programs like Media Portal via My Anime 3 and Plex via Plex Plugin have access to your collection no matter where you are.

JMM Desktop

JMM Desktop is a windows based program built with .NET/WPF for managing your anime collection.

Unlike My Anime 3, which is mainly for watching your collection, JMM Desktop provides a lot of advanced features for better manging and optimizing your anime collection. However like My Anime 3, JMM Desktop also allows you to watch your anime collection and will keep track of anything you watch.

While JMM Server automatically processes your collection, JMM Desktop is considered the front-end of JMM Server providing you with the ability to further fine tune your collection with a wide array of tools to make sure your collection is setup the way you want it to be. With JMM Desktop you can also view your entire collection however you'd like as well as view detailed information about the various series and files within it. 1

My Anime 3

My Anime 3 is a plugin designed for MediaPortal

With My Anime 3, you can easily watch your anime collection on your home theater PC. It displays detailed information not only about your collection, but the different anime series within your collection as well. Fanart, DVD Covers, posters, and banners are automatically downloaded giving you a pleasing look to your anime collection. Each series is listed in full detail showing each episode summary, preview picture and air date.

Learn more about My Anime 3

Plex Plugin

Plex Plugin is designed for use with Plex.

With Plex Plugin, you can stream your anime collection to any device that supports Plex allowing you to always have access to your collection no matter where you go. Like the My Anime 3 plugin for MediaPortal, Plex Plugin is capable of displaying your collection data, including images in a detailed and easy to navigate layout providing you the best experience possible.

Click here to learn more about Plex Plugin

Anime Buddy

Anime Buddy is a Windows Store app available on Windows 8/8.1/10 that when connected to JMM server gives you the ability to stream your anime collection or play it locally. It pulls all relevant information from JMM server allowing you to continue watching where you left out and provides detailed information about any show you select such as the characters in it and what other's have to say.
