What Is JMM

JMM (Japanese Media Manager) is an anime catalog program that automatically catalogs your anime collection regardless of size and without the hassle of having to manually input each and every file in your collection. In addition, JMM provides you with the most flexibility in managing your collection requiring minimal input from the user allowing you to enjoy their collection.

JMM is currently composed of the following programs and plugins.

  • JMM Server - Shared server component
  • JMM Desktop - Windows application
  • My Anime 3 - A plugin for MediaPortal (Home Theatre PC's)
  • Plex Plugin - A plugin for Plex (Home Theatre PC's)
  • AnimeBuddy - A Windows Store app for Windows 8/8.1/10

How does JMM work?

Metadata from various data gathering sites such as AniDB, The TV Database and The Movie Database is gathered and complied for use in JMM to add detailed information about the files and series in your collection which in turn allows JMM to deliver you the best anime experience possible. The main goal of JMM is to reduce the amount of work you have to by making everything as automated as possible. It's cumbersome and boring cataloging hundreds if not thousands of files so let JMM do it for you. This time saving process is possible thanks to AniDB, one of the most respected anime information repository on the internet and it's vast database of anime related data. JMM is able to compare your files without having to rely on the filename by comparing it against AniDB's database which results in your files being recognized, cataloged and immediately ready for playback with absolutely no additional work on your part. JMM also includes an array of advanced tools to identify and catalog any file that in rare instances' it's unable to automatically process.

While there are other anime catalog programs out there that provide similar functionality, there's nothing else like JMM available as JMM uses a client-server based architecture as it's base while other programs just use a client based architecture. What this means is that JMM is more than just a single program or plugin. Using JMM Server, the heart of JMM provides access to your entire collection across different clients such as JMM Desktop and Anime Buddy and media center programs like Media Portal via My Anime 3 and Plex via Plex Plugin. It doesn't matter if you're at home or away, as long as you have an internet connection, your collection is never more than a few clicks away!

Features

Here are some of the key features JMM provides.

  • Client-server based architecture allowing for multiple clients to access your collection
  • Ability to store your collection on both internal and external media, JMM complies it into it's database and keeps track of it all
  • Ability to have multiple user profiles with their own collection, statistics and permissions
  • Advanced filtering and sorting options allows you to display and update your collection data however you want
  • Information repositories such as AniDB, The TV Database and The Movie Database supported for gathering data.
  • Support for MyAnimeList (MAL) and Trakt.TV for importing and exporting your collection statistics.
  • Additionally for Trakt.TV users, support for viewing what your friends are watching.
  • Supports three different database types, SQLite, MySQL and Microsoft SQL Server.
  • Full integration with uTorrent for keeping your collection up-to-date (Additional client support coming soon!)
  • Support for multiple torrent trackers (Anime Byt.es, BakaBT, Nyaa and Tokyo Toshoka) for downloading
  • Support for advanced file management such as file renaming, missing episodes check, duplicate files check, AVDump and more!
  • Ability to generate anime recommendations based off your ratings

Again these are just some of the features provided by JMM, take a look at our documentation for a more detailed look.

JMM Server

JMM Server is the heart of JMM and it's client-server architecture design.

JMM Server handles all the behind the scenes work automating it all so you don't have to manually process your collection. JMM Server performs file hashing, file comparing, information compiling and image downloading for each file you add and it's corresponding series. JMM Server is what allows you to have multiple clients such as JMM Desktop and Anime Buddy and media center programs like Media Portal via My Anime 3 and Plex via Plex Plugin have access to your collection no matter where you are.

Learn more about JMM Server

JMM Desktop

JMM Desktop is a windows based program built with .NET for managing your anime collection.

While JMM Server automatically processes your collection, JMM Desktop is considered the front-end of JMM Server providing you with the ability to further fine tune your collection with a wide array of tools to make sure your collection is setup the way you want it to be. With JMM Desktop you can also view your entire collection however you'd like as well as view detailed information about the various series and files within it.

Learn more about JMM Desktop

My Anime 3

My Anime 3 is a plugin designed for use with MediaPortal

My Anime 3 displays detailed information not only about your collection, but the different anime series within your collection as well and automatically downloads Fanart, DVD Covers, posters, and banners giving you a pleasing look to your anime collection. In addition,each series is listed in full detail showing each episode summary, preview picture and air date and more!

Learn more about My Anime 3

Plex Plugin

Plex Plugin is a plugin designed for use with Plex.

With Plex Plugin, you can stream your anime collection to any device that supports Plex allowing you to always have access to your collection no matter where you go. Like the My Anime 3 plugin for MediaPortal, Plex Plugin is capable of displaying your collection data, including images in a detailed and easy to navigate layout providing you the best experience possible.

Learn more about Plex Plugin

Anime Buddy

Anime Buddy is a Windows Store app available on Windows 8/8.1/10 that when connected to JMM Server gives you the ability to stream your anime collection or play it locally. It pulls all relevant information from JMM Server allowing you to continue watching where you left off and provides detailed information about any show you select providing you with a fantastic experience.

Learn more about Anime Buddy
