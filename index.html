Japanese Media Manager

What is JMM

JMM (Japanese Media Manager) is a series of programs and plugins that allow you to not only watch, but manage your anime collection with a great deal of flexibility. JMM is currently composed of the following programs and plugins.

  • JMM Server - Shared server component
  • JMM Desktop - Windows application
  • My Anime 3 - A plugin for MediaPortal (Home Theatre PC's)
  • Plex Agent - An agent for Plex
  • AnimeBuddy - A Windows Store app for Windows 8/8.1/10

Information from various sites such as AniDB and TheTvDB and TMDb is gathered and complied to deliver the best anime experience possible. The goal of JMM is to reduce the amount of work you have to do by making everything as automated as possible. This is possible thanks to AniDB and it's vast anime database as AniDB is used by most subbing groups as an information repository, which then allows JMM to use this information for your collection. This allows JMM to not have to rely on your files being named in a specific format which results in your files being recognized, cataloged and ready for playback with no additional work on your part. JMM also provides tools to identify and catalog your files in those rare instances where it could not be done automatically.

While there are other programs out there with similar functionality, JMM differs from them as it uses a client-server based architecture called JMM Server. This means you have one central server running which keeps track of your collection, while clients like JMMDesktop, MyAnime3 and Plex Agent connect to the server to display your collection.

Key Features

  • One server allowing for multiple clients
  • Store your anime on internal and external media, JMM complies the information into it's database
  • Create multiple profiles each with their own collection and stats
  • Advanced filtering and sorting options for displaying and updating your collection
  • Supports popular sites AniDB, The TvDB, MovieDB, MyAnimeList and Trakt.tv
  • Supports Sqlite, MySQL or Microsoft SQL Server for your database
  • Direct integration with uTorrent and multiple torrent trackers (Anime Byt.es, BakaBT, Nyaa and Tokyo Toshoka)
  • Support for advanced file management such as file renaming, missing episodes check, duplicate files check, manual linking, AVDump and more
  • Rate your series to get recommendations based off your ratings.
  • For Trakt.tv users, see what your friends are watching

JMM Server

JMM Server is the heart of JMM.

It's responsible for all the behind the scenes work you don't have to do. JMM Server performs all the file hashing, file comparing, information compiling and image downloading and stores it all in it's shared database. With JMM Server, you can run multiple clients such as JMM Desktop or My Anime 3 across multiple computers and always keep your information centralized.

JMM Desktop

JMM Dekstop is a windows based program built with .NET/WPF for managing your anime collection.

Unlike My Anime 3, which is mainly for watching your collection, JMM Desktop provides a lot of advanced features for better manging and optimizing your anime collection. However like My Anime 3, JMM Desktop also allows you to watch your anime collection and will keep track of anything you watch.

My Anime 3

My Anime 3 is a plugin designed for MediaPortal

With My Anime 3, you can easily watch your anime collection on your home theater PC. It displays detailed information not only about your collection, but the different anime series within your collection as well. Fanart, DVD Covers, posters, and banners are automatically downloaded giving you a pleasing look to your anime collection. Each series is listed in full detail showing each episode summary, preview picture and air date.

Plex Agent

Plex Agent is a plugin for Plex that allows you to stream your anime collection that's been cataloged via JMM Server. Like My Anime 3 plugin for MediaPortal, it displays detailed information not only about your collection and the different anime series within your collection as well. Fanart, DVD Covers, posters, and banners are automatically downloaded giving you a pleasing look to your anime collection. Each series is listed in full detail showing each episode summary, preview picture and air date.

Anime Buddy

Anime Buddy is a Windows Store app available on Windows 8/8.1/10 that when connected to JMM server gives you the ability to stream your anime collection or play it locally. It pulls all relevant information from JMM server allowing you to continue watching where you left out and provides detailed information about any show you select such as the characters in it and what other's have to say.
