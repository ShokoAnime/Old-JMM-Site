Installing and Setting Up JMM Server

JMM Server is a Mandatory Install as it's the heart of JMM and is what every program and plugin uses to display your anime collection. This is the first thing you should install to avoid any potential problems.

Once JMM Server has finished downloading, run the installer by double-clicking it and follow all on-screen prompts to install JMM Server. After JMM Server has finished installing you'll need to run it to preform the initial database setup.

You'll be asked to select the database type to use with JMM Server, however before making a choice please read below about the different database types.

At this time, JMM Server supports the following types of database back-ends.

  • SQLite
  • MySQL 5.6+
  • SQL Server 2008+

SQLite is the easiest to install and simply requires you to click Save and Start Server. However SQLite is not designed to properly handle large collections containing 200 series or more. Keep in mind that you cannot convert your SQLite database to MySQL or SQL Server if you decide to switch at a later point.

MySQL and SQL Server are both designed to handle and maintain large databases which make them ideal if your collection is already pretty big or if you intend for your collection to grow. However they are not as easy to install as SQLite and require some additional work. If you want to install either of them then look under Install Instructions in the menu on the left.

Configuring JMM Server With MySQL

For the Database Type, select MySQL.

You'll be asked to input the following information, this is the same information you inputted when you installed MySQL.

  • Server Address - Only change if JMM Server is on a different computer. Default: Localhost
  • Database Name - Only change if you created a custom database for JMM. Default: JMMServer
  • Username - If you created a account just for JMM, use that username. Default: root
  • Password - If you created a account just for JMM, use that username, otherwise it's the root password you inputted.

If everything was inputted correctly JMM Server will connect with MySQL, create the database and populate it.

Congratulations! You've successfully configured JMM Server with MySQL.

Configuring JMM Server With SQL Server

For the Database Type, select SQL Server.

You'll be asked to input the following information, this is the same information you inputted when you installed MySQL.

  • Server Address - Only change if JMM Server is on a different computer. Default: Localhost
  • Database Name - Only change if you created a custom database for JMM. Default: JMMServer
  • Username - Input the username for the account you created during setup. Default: sa
  • Password - Input the password for the account you created during setup.

Pressing the Refresh button will get a list of computers running SQL Server on your local network which you can select from if you don't know the Server Address.

If everything was inputted correctly JMM Server will connect with SQL Server, create the database and populate it.

Congratulations! You've successfully configured JMM Server with MySQL.

JMM Server has been successfully installed and setup. We recommend you take a look around the program before moving on with the documentation. However for those who want to follow this documentation then head over to the JMM Server Index and follow the link order.
