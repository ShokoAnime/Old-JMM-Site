Installing and Setting Up JMM Server

JMM Server is a Mandatory Install and must be installed, we recommend you install it first to avoid any potential problems.

Once JMM Server has finished downloading, run the installer by double-clicking it. Follow all on-screen prompts to install JMM Server. You'll be asked to select a database to use, however before making a selection read below about the different database types.

At this time, JMM Server supports the following types of database back-ends.

  • SQLite
  • MySQL 5.6+
  • SQL Server 2008+

SQLite is the easiest to install and simply requires you to click Save and Start Server. However SQLite is not designed to properly handle large collections containing 200 series or more. Keep in mind that you cannot convert your SQLite database if you decide to switch at a later point. If you want to install MySQL or SQL Server instead, use the links in the menu on the left.

JMM Server is now installed! Now it's time to learn how to use JMM Server.
