Installing and Setting Up Anime Buddy

Open up the Windows Store and search for Anime Buddy.

Once you've found it, click the Install button. Once it's done installing start Anime Buddy.

By default, Anime Buddy will only show you a calendar of upcoming TV Shows, Movies, OVA's and Adult Titles. To have it show files from your collection you'll need to connect Anime Buddy to JMM Server. You can do this by clicking Login to JMM in the upper-right corner.

You'll be asked to input some information Anime Buddy needs to connect to JMM Server.

If JMM Server and Anime Buddy are on the same computer, use the following information

JMM Server Address: Use either 127.0.0.1 or localhost
JMM Server Port: By default it's 8111
Username: Input your JMM Desktop Username
Password: Input your JMM Desktop Password, if blank than leave blank

If JMM Server and Anime Buddy are on different computers, use the following information

JMM Server Address: Input the IP address of the computer JMM Server is on
JMM Server Port: By default it's 8111
Username: Input your JMM Desktop Username
Password: Input your JMM Desktop Password, if blank than leave blank

If you've inputted the correct information, Anime Buddy will refresh and your collection will be shown. Make sure to check out the Frequently Asked Questions page for more information.

If you like Anime Buddy, please consider reviewing in on the Windows App Store!
