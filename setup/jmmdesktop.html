Installing and Setting Up JMM Desktop

Running JMM Desktop For The First Time

Before you install and setup JMM Desktop, make sure you'be installed and setup JMM Server, you won't be able to properly install and setup JMM Desktop if you haven't.

Like JMM Server, JMM Desktop is a mandatory install and is what you'll be using to manage your anime collection. Once JMM Desktop has finished downloading, run the installer by double-clicking it and follow all on-screen prompts to install JMM Desktop.

If you attempt to run JMM Desktop without JMM Server already running, you'll be greeted with the following error message.

This just means that JMM Desktop was unable to connect to JMM Server. Click OK and you'll be taken back to a modified version of the JMM Server settings page inside JMM Desktop.

If JMM Server and JMM Desktop are on the same computer, click Test And Save and try again. If you're still receiving the same error, verify JMM Server has been properly setup and running and the Server Address you are using is correct.

If JMM Server and JMM Desktop are on the different computers, you'll need to input the IP Address of the computer JMM Server is on. Visit WhatsMyIP on the computer that JMM Server is installed on to get the IP Address, click Test And Save and try again. If you run into any connection problems, follow the solutions on this page as it covers most connection problems users encounter.

Once you're able to connect, a window will pop-up asking you to select the username and input a password.

When JMM Server first created your database, it created two different user accounts called Default and Family Friendly, by default the password for both accounts was left blank. Here's the difference between the two accounts.

  • Default - Will display all content in your collection.
  • Family Friendly - Will only display content from your collection that is family friendly, all other content is hidden.

We recommend you initially select the Default account so you can configure JMM Desktop. Once you've logged in, you can easily make changes to the default accounts, remove them or add additional accounts by clicking on the Settings tab and then clicking the User section.

The Left Panel allows you to Add or Remove users while the Right Panel allows you to configure the options for each user.

  • Username - The username for the account, it can be anything you want it to be.
  • Administrator - If check the account with had administrator access, they can edit everything but server settings.
  • AniDB User - Will tell JMM Server to upload their metadata to AniDB for the user.
  • Trakt User - Will tell JMM Server to upload their metadata to Trakt for the user.
  • Edit Server Settings - Can edit JMM Server settings.
  • Hide Tags - Tags added to this list will hide any series that has any added tag.

Located at the bottom is the Change Password button you can use to set a password if you're using one of the created user accounts. A password is not needed or required but is great if there are multiple users with varying levels of permission using JMM Desktop.

Once down click the Save button at the bottom.

Now head over to Essential section which is located in the Settings tab, this is where all your login information for AniDB is stored and your import settings. You can also edit your JMM Server settings here as well allowing you to make changes as needed all within JMM Desktop.

If you've been following this guide then a lot of the information has already been inputted. We recommend that you double-check your AniDB info and your import folders before configuring your import settings. If you plan to use AvDump2, you'll also need to input API Key before running AVDump2.

At the bottom of the Essential section are the import settings. Take a moment to look over each setting before moving on as this will determine what gets added to your collection and how it gets added.

  • Video Extensions - Any video extension on this list will be included during JMM Server's scanning of your files.
  • Automatically Place Related Series Into The Same Group - If selected, series with OVA's and Movies will be put into the same group.
  • Import On Start - Will run the import process every time JMM Desktop is started.
  • Calculate CRC32 - Will calculate the CRC32 value.
  • Calculate MD5 - Will calculate the MD5 value.
  • Calculate SHA1 - Will calculate the SHA1 value.
  • Images Path - The location JMM Server will use when downloading images for series in your collection.

Congratulations! JMM Desktop has been successfully installed and setup. We recommend you take a look around the program before moving on with the documentation, you'll be able to better learn the program through trial and error. However for those who want to follow this documentation then head over to the JMM Desktop Index and follow the link order.
