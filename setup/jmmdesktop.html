Installing and Setting Up JMM Desktop

JMM Desktop is a Mandatory Install and must be installed, we recommend you install it after installing JMM Server to avoid any potential problems.

If you start JMM Desktop without having JMM Server running or if JMM Server is located on a different computer, you'll see the following error message.

Clicking OK will take you back to the JMM Server settings page

If JMM Server is running on the same computer simply start JMM Server then click Test And Save once JMM Server has finished loading your database.

If JMM server is running on a different computer then input the IP address of the computer it's on. If on the same network then use the network IP address. If on a computer not on your network enter the computer's IP address. Once down click Test And Save, if correct JMM Desktop will start.

If it connects successfully or if you are running JMM Desktop and JMM Server on the same machine you'll see the login prompt.

When JMM Server created your database, it created two different users labeled Default and Family Friendly, by default the password was left blank. Here's the difference between the two user accounts.

  • Default - Will display all your content.
  • Family Friendly - Has hidden content based on the categories/genres that are not family friendly

If you'd like to make changes to the default users you can once you're done setting up JMM Desktop by clicking on the Settings tab and then clicking the User section.

Now before you do anything else click the Settings tab and then the Essential section, you need to input some additional information for JMM Desktop and JMM Server to work properly.

The Essential section is where all information required for JMM Server and JMM Desktop to work is stored allowing you easy access to make changes as needed.

If you've been following this guide then you may of seen the AniDB section before on JMM Server. The first thing you need to do is input both your AniDB username and password. An AniDB account is required for JMM Server and JMM Desktop to work, Click Here to sign up. Once done click Test and Save to make sure your inputted AniDB information is valid. If you encounter any errors double-check your account information is correct by logging into the AniDB website.

Now setup your Import Folders, you just need to add the root folder of the location where you keep your anime. All sub folders are automatically added. If JMM Desktop is on a different computer than JMM Server, you will need to enter the path as it is on the server, not on the client. The screenshot below has an example of a mapped server path.

You can ignore the other settings for now, for the last part click the Server tab. The Server tab shares a lot of options with JMM Server, so if you need to make changes you can do it via JMM Server as well.

Click the Run Import button, this will start scanning and adding your anime into your collection. Depending on the amount of files that need to be scanned this process could take a very long time. JMM Desktop does not need to be running as the importing process is handled by JMM Server. However if you'd like to view your collection while it's importing click on the Collections tab to see what's been imported so far.

JMM Desktop is now installed and setup!

If you need to install and setup anything else, click it's related link on the menu on the left.
