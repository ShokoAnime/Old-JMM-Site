Installing and Setting Up Plex Plugin

Installing the Plex plugin is pretty simple, copy MyAnime.bundle to your Plex plugins folder and you're done. You can do this by right-clicking on the Plex server tray icon, and selecting Open Plug-ins folder.

It will open the plugins folder allowing you to copy MyAnime.bundle into it. If asked to over-write or if Windows' UAC pops up click OK.

Once you've copied the plug-in over, you'll need to restart Plex so it can load the plug-in.

Click on the taskmenu and select Exit, this will close Plex winch then will allow you to restart it.

The Plex plugin has been installed and will appear under Channels as My Anime.

Clicking the My Anime channel will show your collection allowing you to view series, episode and character details in addition to streaming your collections files.

The Plex plugin is now installed!

If you need to change any of the settings the Plex Plugin uses to connect to JMM Server. Open the plug-in Settings by pressing the cog in the top-right or the cog on the plug-in logo in the channel menu. This will bring up the following window.

Here's a breakdown of the available options.

  • Server IP - The IP Address of the computer JMM Server is on.
  • Port - The Port JMM Server is using.
  • JMM User - The account to use when connecting.
  • Search Limit - The max amount of items to display when searching.

If Plex and JMM Server are on the same computer, you can use localhost or 127.0.0.1 as the Server IP.
