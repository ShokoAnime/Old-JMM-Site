Installing and Setting Up MySQL

MySQL is the recommend database type for use with JMM Server for users with a large collection, who intend to build a large collection or simply for any user that wants better and faster performance with the JMM suite.

Why MySQL over SQLite

If your collection contains more than 200 series, performance wise your better off using MySQL. While SQLite is a one click setup single file, it's not designed to handle a database that size and it will show it's limitations as your collection continues to grow by causing slowdowns when accessed by JMM.

Here are some of the differences between MySQL and SQLite

  • MySQL is more powerful but will use more memory than SQLite.
  • SQLite does not handle multiple operations as well as MySQL.
  • Using SQLite and running any kind of tasks in JMM Server will cause considerable and noticeable slowdowns when running multiple JMM products.

Download

There are a lot of different options available when browsing the downloads section on the MySQL site. What you want to download is the latest version of MySQL Community Server.

Installing MySQL Community Server

warning:
There is a known error with MySQL Community Server that will prevent you from updating during the install process. If there is a space in your Windows Account Name you'll be unable to update MySQL Community Server. The fix can be found here in the following post: [7 Aug 9:13] Jonathan Meier.

The following instructions were written using MySQL Community Server Version 5.6.20.0. While newer versions should not differ too much, make sure the items you're installing match up with the items in the following images.

Start the installer and if prompted download any available updates. Once done select the Custom install option.

Some versions of the MySQL Community Server do not allow you to change the install and data path so don't worry if those options are not present.

Follow the following pictures selecting the same options. If presented with both 32-bit and 64-bit options, install the one that corresponds with your computer setup or both if you don't know your computer setup.

The install will check to make sure you have all prerequisites installed that are required to run MySQL Community Server. Once all prerequisites have finished downloading and have been install click the Next button.

The installer automatically downloads and installs all selected products, depending on your connection speed this could take some time. If anything fails when downloading or installing don't worry, just keep clicking the retry button until it succeeds.

Configuring MySQL Community Server

Now that MySQL Community Server has been downloaded and installed successfully it's now time to configure it for use.

Development Machine is the recommended Server Configuration Type though advanced users might want to check out some of the other options.

You'll now be prompted to set the Root Account Password, how easy or hard you make it is up to you but make sure to write it down. You'll need to it for JMM Server to create and connect to your database. By default JMM Server uses root as the default username and the password you inputted. However you can create a MySQL user specifically for JMM Server if you plan to use MySQL with additional applications.

It's recommended you have MySQL Server start with Windows. Otherwise you will need to manually run it each time you want to use any JMM product.

Congratulations! MySQL Server has been successfully installed! If you've paused your JMM Setup, you can now continue with the install.
