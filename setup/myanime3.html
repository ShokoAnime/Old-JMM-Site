Installing and Setting Up My Anime 3

After downloading My Anime 3 you need to install it, the MediaPortal has written a detailed guide on how to install plugins which we recommend you read to install My Anime 3.

Once you've installed My Anime 3 it's time to set it up. Run the MediaPortal configuration and click on Plugins.

Right click on My Anime 3 and select Configuration. For quick access to My Anime 3, while viewing the properties menu select Listed in Home to have My Anime 3 show up on the home screen.

The default settings should work if JMM Server and MediaPortal are on the same computer. If they are on separate computers you'll need to change the Server Address to match the computer JMM Server is on.

My Anime 3 is now installed and setup!
