Installing and Setting Up My Anime 3

Once you've downloaded My Anime 3, you'll need to install the plugin. We're going to skip over instructing you on how to install the plugin as the MediaPortal team as written a very detailed guide on installing plugins which we recommend you use instead to install My Anime 3.

After you've finished installed My Anime 3, you'll need input your settings so it will work properly. To do this run the MediaPortal Configuration and select Plugins from the list of options.

You'll see a list of your installed plugins, right-click on My Anime 3 and select Configuration, this will open up the settings menu. Additionally, you can add My Anime 3 to your home screen by selecting Listed in Home from the list of options.

If JMM Server and MediaPortal are on the same computer then the default settings should work. However if JMM Server and MediaPortal are on separate computers, you'll need to change the Server Address and Port to match the computer JMM Server is on. If you're trying to connect to JMM Server that's on another computer either on your network or over the internet, the following instructions provide detail information for setting everything up as well as solutions to the most common problems.

With that done, you've successfully setup and installed My Anime 3! You don't need to change any of the other settings as they are imported from JMM Server. However we recommend you take a look around at the other settings to better familiarize yourself with My Anime 3.
